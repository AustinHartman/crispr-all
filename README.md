# crispr-all

Code to reproduce plots in the CRISPR-All-Seq [prepint](https://www.biorxiv.org/)

## Description of contents
* `01_base_screen_plots.R` contains code to generate plots in Figure 4
* `02_base_library_fidelity_plots.ipynb` contains scripts to generate plots in Figure 4
* `03_scRNA_plots.R` contains code to generate single-cell plots in Figure 5
* `04_combo_screen_plots.R`
* `05_combo_library_fidelity_plots.ipynb`
